# BUG-004 — 레이아웃 편집 기능 수정사항

- `src/pages/LayoutEditorPage.tsx`

- `src/components/layout-editor/EditorToolbar.tsx`
- `src/components/layout-editor/LayerPanel.tsx`
- `src/components/layout-editor/PropertiesPanel.tsx`
- `src/stores/layoutEditorStore.ts`

---

## 수정 1 — 배치 영역 더블클릭 시 명칭 수정

**현재 상태:**
- 배치 영역 생성 후 명칭 수정 불가

**변경 요구사항:**
- 배치 영역 더블클릭 시 명칭 수정 가능하도록 인라인 편집 활성화

---

## 수정 2 — 단계별 그리기 도구 자동 활성화

**현재 상태:**
- 배치 영역 지정 / OHT 레일 그리기 단계에서
  사용자가 직접 도구를 선택해야 함

**변경 요구사항:**
- 설비 배치 영역 지정 단계 진입 시
  자동으로 배치 영역 그리기 도구 활성화
- OHT 레일 그리기 단계 진입 시
  자동으로 OHT 레일 그리기 도구 활성화
- 사용자가 도면에 클릭만 하면 바로 그리기 시작

---

## 수정 3 — 상단 속성 버튼 추가

**현재 상태:**
- 생성한 개체들의 명칭을 한눈에 볼 수 있는 기능 없음

**변경 요구사항:**
- 편집 화면 상단에 `속성` 버튼 추가
- 속성 버튼 클릭 시 현재 단계에 해당하는 개체 목록을 표로 표시
- 설비 배치 영역 지정 단계에서 속성 버튼 클릭 시:
  - 생성된 배치 영역 목록을 표로 제공
  - 표에서 배치 영역 명칭 일괄 수정 가능
  - 행 정렬 순서: 사용자가 생성한 순서 기준

---

## 수정 4 — 설비 레이어 컬럼 표기 오류

**현재 상태:**
- 컬럼 선택 시 컬럼명이 아닌 컬럼 키(예: `col_1775254369604`)로 표기됨

**변경 요구사항:**
- 컬럼 선택 드롭다운에서 컬럼 키 대신 **사용자가 지정한 컬럼명**으로 표기

---

## 수정 5 — 설비 레이어 컬럼 매칭 순서 변경

**현재 상태:**
- 컬럼 매칭 순서: Xmax → Xmin → Ymax → Ymin → EQP_ID

**변경 요구사항:**
- 컬럼 매칭 순서: **EQP_ID → Xmax → Xmin → Ymax → Ymin**

---

## 수정 6 — 설비 생성 버튼 추가 및 도면 렌더링

**현재 상태:**
- 컬럼 매칭 완료 후 도면에 설비가 표시되지 않음

**변경 요구사항:**
- 컬럼 매칭 완료 후 `설비 생성` 버튼 추가
- 버튼 클릭 시 지정한 좌표 컬럼(Xmax/Xmin/Ymax/Ymin) 기준으로
  도면 위에 설비 객체 렌더링
- 데이터가 변경될 때마다 버튼을 다시 클릭하여 도면 갱신 가능
- 사내망 이관 시 교체될 수 있도록 데이터 호출부에 주석 표기:
```typescript
// [사내망 이관 시 교체] 설비 데이터 실제 API로 교체 필요
```

---

## 수정 7 — 시설물 레이어 명칭 컬럼 매칭 추가

**현재 상태:**
- 시설물 레이어 컬럼 매칭에 좌표 컬럼만 존재
  (Xmax / Xmin / Ymax / Ymin)

**변경 요구사항:**
- 명칭 컬럼 매칭 항목을 **가장 상단**에 추가
- 매칭 순서: **명칭 → Xmax → Xmin → Ymax → Ymin**

---

## 수정 8 — 보기 모드 도면 세로 폭 전체 화면으로 변경

**현재 상태:**
- 보기 모드에서 도면의 세로 폭이 화면 절반 크기로 표시됨

**변경 요구사항:**
- 도면 영역이 화면 전체 세로를 차지하도록 수정
