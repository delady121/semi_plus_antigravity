## 버그 1 — 보기 모드 좌클릭 드래그 범위 선택 미작동

**현재 상태:**
- 보기 모드에서 좌클릭 드래그로 범위 사각형 생성 후
  범위 안의 설비 다중 선택이 작동하지 않음

**변경 요구사항:**
- 보기 모드에서 빈 공간 좌클릭 드래그 시
  반투명 파란색 범위 선택 사각형 생성
- 드래그 종료 시 사각형 범위 안에 포함된
  현재 클릭 반응 설정 레이어의 개체 다중 선택
- 선택된 개체는 강조 표시

---

## 버그 2 — 설비 레이어 반투명 제거

**현재 상태:**
- 설비 객체가 반투명하게 표시됨

**변경 요구사항:**
- 설비 객체 불투명도(opacity) 100%로 변경
- 반투명 효과 완전 제거

---

## 버그 3 — 설비 위치 조정 시 데이터 테이블 좌표 최신화

**현재 상태:**
- 초기설정 편집 모드에서 설비를 드래그로 이동해도
  연결된 데이터 테이블의 좌표값이 업데이트되지 않음

**변경 요구사항:**
- 초기설정 편집 모드에서 설비 드래그 이동 완료 시
  연결된 데이터 테이블의 Xmax / Xmin / Ymax / Ymin 값 자동 업데이트
- 보기 모드에서는 설비 드래그 이동 완전 비활성화
  (클릭 선택만 가능, 이동 불가)
- 사내망 이관 시 교체될 수 있도록 좌표 업데이트 로직에 주석 표기:
```typescript
// [사내망 이관 시 교체] 설비 좌표 업데이트 실제 API로 교체 필요
```
