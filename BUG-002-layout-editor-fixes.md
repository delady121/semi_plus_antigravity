## 버그 1 — 배치 영역 / OHT 레일 편집 기능 부재

**현재 상태:**
- 배치 영역 및 OHT 레일을 한번 그리면 이후 수정이 불가능함

**변경 요구사항:**

**① 그리기 방식**
- 좌클릭으로 꼭지점을 찍어가며 그리기
- 우클릭으로 그리기 완료
- 그리기 완료 시 해당 개체의 정보 입력 팝업 표시

**② 편집 기능**
- 삭제
- `Ctrl+Z` 실행취소
- 꼭지점 드래그로 크기 조정
- 드래그로 위치 이동
- `Ctrl+C` / `Ctrl+V` 복사 + 붙여넣기

**③ 데이터 연동**
- 생성된 배치 영역 및 OHT 레일 데이터를 **데이터 관리 메뉴에서 조회** 가능
- 데이터 관리에서 컬럼 추가를 통해 각 영역/레일별 관련 정보 입력 가능

## 버그 2 — 설비 레이어 테이블 선택 시 잘못된 데이터 참조

**현재 상태:**
- 설비 레이어 설정에서 테이블 선택 시
  데이터 관리에 존재하는 테이블이 아닌
  별도의 목업 데이터를 바라보고 있음
- 데이터 관리에서 생성/관리하는 테이블과 전혀 무관한 데이터가 표시됨

**변경 요구사항:**
- 설비 레이어 테이블 선택 목록이
  **데이터 관리에 실제로 존재하는 테이블 목록**을 참조하도록 수정
- 데이터 관리에 테이블이 없을 경우:
  "데이터 관리에서 테이블을 먼저 생성해주세요" 안내 문구 표시
- 사내망 이관 시 실제 API로 교체될 수 있도록
  데이터 호출부에 아래 주석 표기:

```typescript
// [사내망 이관 시 교체] 데이터 관리 테이블 목록 실제 API로 교체 필요
```
