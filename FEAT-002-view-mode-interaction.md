## 기능 1 — 보기 모드 마우스 인터랙션 변경

**현재 상태:**
- 보기 모드에서 마우스 인터랙션이 명확하게 정의되어 있지 않음

**변경 요구사항:**

**① 좌클릭 — 선택 기능**
- 개체 클릭 시 해당 개체 선택
- 빈 공간에서 드래그 시 범위 선택 사각형 생성
- 사각형 범위 안에 포함된 개체 다중 선택

**② 우클릭 — 지도 이동 기능**
- 우클릭한 상태에서 마우스 드래그 시 도면 이동 (패닝)
- 드래그 중 커서 모양 변경 (grab → grabbing)

---

## 기능 2 — 클릭 반응 레이어 설정

**현재 상태:**
- 클릭 시 어떤 레이어의 개체가 선택될지 설정 기능 없음

**변경 요구사항:**
- 도구 모음에 `클릭 반응 설정` 버튼 또는 패널 추가
- 라디오 버튼 방식으로 한 번에 하나의 레이어만 선택 가능
- 선택된 레이어의 개체만 좌클릭으로 선택 가능
- 선택 대상 레이어 목록:
  - 설비
  - 시설물
  - 배치 영역
  - OHT 레일
  - 사용자 정의 레이어 (생성된 레이어가 있을 경우 목록에 추가)
- 기본값: 설비 레이어 선택 상태
