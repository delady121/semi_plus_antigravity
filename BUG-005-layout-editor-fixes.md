## 수정 1 — OHT 레일 더블클릭 명칭 수정

**현재 상태:**
- 배치 영역 더블클릭 시 명칭 수정 기능이 구현되어 있으나
  OHT 레일에는 동일한 기능이 적용되지 않음

**변경 요구사항:**
- OHT 레일 더블클릭 시 배치 영역과 동일하게 명칭 수정 가능하도록 적용

---

## 수정 2 — 속성 버튼 단계별 표시 내용 분리

**현재 상태:**
- 속성 버튼 클릭 시 어느 단계에서나 배치 영역 목록만 표시됨

**변경 요구사항:**

**① 단계별 표시 내용 분리**
- 설비 배치 영역 지정 단계: 생성된 배치 영역 목록 표시
- OHT 레일 그리기 단계: 생성된 OHT 레일 목록 표시

**② #컬럼 셀 클릭 시 해당 개체 강조**
- 속성 테이블의 #컬럼(순번) 셀 클릭 시
  해당 배치 영역 또는 OHT 레일이 캔버스에서 강조 표시

**③ 데이터 관리 테이블 자동 생성**
- 생성된 배치 영역 및 OHT 레일 데이터가
  데이터 관리 메뉴에 테이블로 자동 생성
- 테이블 명 규칙:
  - 배치 영역: `{레이아웃명}_배치영역`
  - OHT 레일: `{레이아웃명}_OHT레일`
- 사내망 이관 시 교체될 수 있도록 데이터 저장부에 주석 표기:
```typescript
// [사내망 이관 시 교체] 데이터 관리 테이블 저장 실제 API로 교체 필요
```

## 수정 3 — OHT 레일 그리기 스냅핑 기능 추가

**현재 상태:**
- OHT 레일 그리기 시 스냅핑 기능 없음

**변경 요구사항:**
- OHT 레일 그리기 시 기존 OHT 레일 및 배치 영역 꼭지점에 스냅핑
- 스냅핑 ON/OFF 토글 버튼 도구 모음에 추가

---

## 수정 4 — 시설물 레이어 시설물 생성 버튼 추가

**현재 상태:**
- 설비 레이어에는 설비 생성 버튼이 있으나
  시설물 레이어에는 동일한 버튼이 없음

**변경 요구사항:**
- 시설물 레이어 컬럼 매칭 완료 후 `시설물 생성` 버튼 추가
- 버튼 클릭 시 지정한 좌표 컬럼 기준으로 도면 위에 시설물 객체 렌더링
- 데이터 변경 시 버튼 재클릭으로 도면 갱신 가능
